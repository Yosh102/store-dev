# PLAY TUNE STORE

## プロジェクト概要

PLAY TUNEの公式ストアプロジェクトです。ストア機能、ファンクラブ機能をメインに実装予定です。

## 現在の実装状況

### 実装済み機能

1. **認証**
   - Firebase Authを使用したメール/パスワード認証とGoogle認証
   - ユーザーロール（一般ユーザー、管理者）

2. **ユーザー管理**
   - ユーザープロフィール管理（表示名、メールアドレス、プロフィール画像）
   - Firestoreを使用したユーザーロール管理

3. **ストア機能**
   - 商品一覧と詳細ページ
   - ショッピングカート機能
   - Stripeを使用した決済プロセス

4. **管理者ダッシュボード**
   - 基本的な管理者認証とアクセス制御
   - 商品管理（追加、編集、削除）

5. **お問い合わせ機能**
   - Zendeskと連携したお問い合わせ機能を実装
6. **API**
   - 商品の取得とフィルタリング
   - Stripe決済の統合

7. **フロントエンドコンポーネント**
   - Tailwind CSSを使用したレスポンシブデザイン
   - 再利用可能なUIコンポーネント（ボタン、カード、入力フォームなど）

## 技術スタック

- フロントエンド: Next.js（TypeScript）
- バックエンド: Firebase（Auth, Firestore, Storage, Functions）
- 決済処理: Stripe
- カスタマーサポート: Zendesk
- デプロイ: AWS EC2（予定）
